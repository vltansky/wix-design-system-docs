
### dataHook
- type: string
- description: Applies a data-hook HTML attribute that can be used in the tests
### className
- type: string
- description: Specifies a CSS class name to be appended to the component’s root element.
### type
- type: enum
- description: Changes the way avatars are displayed inside the group.
### size
- type: enum
- description: Changes the size of avatars in the group
### showDivider
- type: bool
- description: Separates the first avatar with a divider  from the rest of the group
### maxItems
- type: number
- description: Sets the maximum number of avatars to be displayed. Number which exceeds this limit will be hidden under the “N+” item.
### items
- type: arrayOf
- description: Use to pass an array of avatars
